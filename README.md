# opsdroid skill random

A skill for [opsdroid](https://github.com/opsdroid/opsdroid) to perform random events like flipping a coin or rolling a dice.

## Requirements

None.

## Configuration

None.

## Usage

#### `flip a coin`

Flips a coin.

> user: flip a coin
>
> opsdroid: It landed on heads

#### `roll a dice`

Rolls a dice.

> user: roll a dice
>
> opsdroid: You rolled a 4
